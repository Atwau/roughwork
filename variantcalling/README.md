# Variant Calling Step-by-step

1. Creat virtual environment where to install isolated tools
to perfomr variant calling. This helps to avoid conflicts with system tools
2. Install the necessary tools for variant calling: bwa, bcftools, samtools,
python, biopython etc 
3. Download the data for which you want to perform variant calling. This can
also be your lab generated data. For this exercise, just download data of 
your choice from NCBI
